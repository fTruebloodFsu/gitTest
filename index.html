Git Commands

  • clone: copy a repo to local machine
  • status: gives differences between local and repo
  • git add [filename]: adds the file to your repo on local machine
  • git add -A: adds all changes to your repo on local
  • git commit -m: commit changes to github, with a message, but doesn't sync
  • git push: syncs all commits with GH
  • git pull: takes changes from GH and puts them on local machine
